## 📌 Sobre o projeto

Este projeto é uma **calculadora de consumo elétrico inteligente**, desenvolvida em Python, que permite estimar o consumo mensal de energia de um aparelho com base no seu uso diário.

## 🎯 Objetivo

Ajudar usuários a entenderem melhor o consumo de energia e o custo estimado de seus aparelhos.

## ⚙️ Tecnologias utilizadas

- 🐍 Python
- 💻 Terminal / CLI
- 🌐 GitHub
